Corner Tag Directive

Usually used as a corner mark of the corner.

Example

Key Code

        
            .side-corner-tag{
                float: right;
                width: 110px;
                height: 45px;
                position: absolute;
                bottom: 0;
                right: -45px;
                /*use parent font-size*/
                /*font-size: 16px;*/
                text-align: center;
                transform: rotate(-45deg);
                -webkit-transform: rotate(-45deg);
                -moz-transform: rotate(-45deg);
                -o-transform: rotate(-45deg);
                -ms-transform: rotate(-45deg);
                color: #fff;
                line-height: 45px;
            }
